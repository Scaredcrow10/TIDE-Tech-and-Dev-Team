# 项目理解文档 - 昝达

## 一、项目目标
这个项目最终要搭建一个基于 AI 的项目管理协作系统，用户在 Mattermost 里用自然语言下达指令，AI Agent 自动创建任务，数据存到后端数据库，前端 Dashboard 做可视化展示，最终能在本地用 Docker 跑通整个流程。

## 二、整体链路
整体流程是：用户在 Mattermost 发消息 → AI Agent 解析意图和关键信息 → 调用后端 API 创建/更新数据 → 数据存入 PostgreSQL → 前端 Dashboard 展示最新状态。所有服务最终通过 Docker Compose 在本地联调运行。

## 三、MVP 范围理解
MVP 是实现从输入到展示的最小可使用版本：用户发消息 → Agent 解析 → 后端存储 → Dashboard 展示。不追求 UI 美观和复杂功能，只是链路跑通。例如，AI Agent 只要能识别“创建一个名为XX的项目”这类简单意图，后续复杂的自然语言处理可以再迭代优化。
